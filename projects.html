Network Traffic IDS

Virtualized lab in Ubuntu to capture live traffic with Wireshark/tshark, then preprocess and feed features into a machine learning model to flag suspicious flows.

Tech: Python, Ubuntu, Wireshark/tshark, ML (e.g. tree-based models)

MATLAB Match-3 Game

A Match-3 style game built in MATLAB with custom sprite handling, cascade logic, scoring, and a simple UI. Focused on clean game loop design and state management.

Tech: MATLAB, GUI, basic game architecture

Java Scoreboard

Console-based scoreboard manager that tracks players/teams using ArrayLists with a menu-driven interface. Emphasizes separation of concerns and readable code.
